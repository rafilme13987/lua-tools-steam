# les option

 1. Direct Download / vous permet de télecharger votre .lua directement depuis une App ID 

 2. Smart Search qui / vas vous permetre de chercher un jeux et de le télécharger directement aprés 

 3. Clear Auth Cache / puisque sa marche avec QWebEngine (libs pyqt6) que on utilise pour intéragire vous pourrez directement sourpimer le cache du navigateur de pyqt6 que je utilise pour se projet

 4. Exit System / vous permet de quitter l'application
