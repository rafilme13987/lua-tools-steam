# Lua-Tools-Steam

Bienvenue sur Lua-Tools-Steam ! Je suis très content de vous présenter ce projet, développé le temps d'un week-end en live coding avec l'aide de Gemini 3.1. 

C'est un outil entièrement codé en Python, conçu pour être simple, efficace et facile à prendre en main. Son interface utilise PyQt6 pour une interaction fluide.

---

## Fonctionnalités

L'application propose un menu clair avec 4 options principales :

* 1. Direct Download : Vous permet de télécharger votre fichier `.lua` directement depuis un App ID Steam.
* 2. Smart Search : Vous offre la possibilité de rechercher un jeu spécifique et de lancer le téléchargement directement après l'avoir trouvé.
* 3. Clear Auth Cache : L'outil fonctionnant avec QWebEngine (via les librairies PyQt6) pour les interactions web, cette option vous permet de vider instantanément le cache du navigateur intégré.
* 4. Exit System : Pour quitter l'application proprement.

---

## Installation & Utilisation

Pour faire fonctionner l'outil, c'est très simple :

1. Clonez ou téléchargez ce repository.
2. Lancez le fichier `start.bat`.
3. Le script se chargera d'installer tous les modules requis (si ce n'est pas déjà fait), puis l'application s'ouvrira automatiquement.
